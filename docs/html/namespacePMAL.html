typedef struct PMAL::ManagerSpec PMAL::ManagerSpec

A struct to provide information to MemoryManger on instantiation.

Once passed to MemoryManager it is fine for the user to destroy this struct. Usage will no longer be needed. Note the total size of memory on the heap is sizeBytes + sizeBytesRegistry however sizeBytesRegistry can be set to zero and meta data will be allocated to the heap at random which will more than likely incure performance overhead.

Parameters
[size]sizeBytes The size of memory to be allocated and managed on the heap for the user.
[size]sizeBytesRegistry The size of registry. Default 1024*1024+16 bytes
[bool]resizeAllowd If the memory being managed turns out to not be sufficient this options lets MemoryManager dynamically resize the memory for extra space. Resizes by 20% by default or what is needed to accomodate the data being constructed.
[float]resizeFactor Factor to increase memory by if needed/allowed.
[bool]success An indicator of the operation of MemoryManager on instantiation (i.e. was the memory successfully allocated). Recommended to be checked before destroying this struct.
[bool]enableStatisitics An option if set to true will enable statisitics to be gathered by MemoryManager. Recommended during debug builds. MemoryStatistics object will track each MemoryManagers stats and collate.
