42 * Once passed to MemoryManager it is fine for the user to destroy this struct. Usage will no longer

46 *
47 * @param [size] sizeBytes

52 * If the memory being managed turns out to not be sufficient this options lets MemoryManager

55 * @param [float] resizeFactor
56 * Factor to increase memory by if needed/allowed.
57 * @param [bool] success

60 * @param [bool] enableStatisitics
